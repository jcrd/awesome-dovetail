Module awesome-dovetail

A tiling layout featuring two visible tiled clients.

Queue the set master command so that the next client to receive focus becomes the master client.

If the focused client is the master, replace it with the first client in the stack. If it's not the master client, make it the master.

If the master client is focused, cycle through clients making each master. If the stack is focused, cycle through clients other than the focused client.
