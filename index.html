Module awesome-dovetail

Functions

get_tag (s)
Override this function to change how dovetail determines the current tag. The function takes the current screen.

Parameters:

  • s The current screen.

Usage:

    dovetail.get_tag = custom_get_tag
layout (layout)
Check if a layout is a dovetail layout.

Parameters:

  • layout The layout or the current layout if nil.

Returns:

    true if layout is a dovetail layout.

focus.byidx (i)
Focus a client in the stack by its relative index.

Parameters:

  • i The index.

Returns:

    true if conditions were met to search for a client. If false, a fallback focus.byidx method should be used.
focus.other ()
If the master client is focused, focus the visible client in the stack, and vice versa.

Parameters:
